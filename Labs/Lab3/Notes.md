## 📡 What is DHCP?

DHCP (Dynamic Host Configuration Protocol) is a network protocol that automatically assigns IP addresses and other network settings to devices.

Instead of manually configuring each device, DHCP does it automatically.

---

## 🔄 DHCP Process (DORA)

1. **Discover** – Device looks for DHCP server
2. **Offer** – Server offers an IP address
3. **Request** – Device accepts the offer
4. **Acknowledge** – Server confirms assignment

---

## ✅ Benefits of DHCP

* Eliminates manual IP configuration
* Reduces errors (IP conflicts)
* Faster network setup
* Centralized management

---

## 🌐 Default Gateway

* The router (192.168.1.1) acts as the default gateway
* It allows devices to communicate outside their network

---

## 📶 Wireless Networking Notes

* SSID = network name
* WPA2-PSK = secure encryption standard
* Strong passwords prevent unauthorized access

---

## 🔁 NAT (Network Address Translation)

* Converts private IPs → public IP
* Allows multiple devices to share one internet connection

---

## 🔍 Troubleshooting Tips

* No IP?
  → Check DHCP is enabled

* Can't connect to WiFi?
  → Check SSID and password

* Can't ping?
  → Check:

  * IP address
  * Gateway
  * Device connection

* Wrong network?
  → Ensure all devices are in 192.168.1.0/24

---

## ⚠️ Common Mistakes

* Using wrong cable type
* Forgetting DHCP
* Wrong WiFi credentials
* Connecting to WAN instead of LAN
