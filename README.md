# 🖧 Cisco Packet Tracer Labs Portfolio

Welcome to my hands-on networking lab repository.

This project showcases practical implementations of core networking and security concepts using Cisco Packet Tracer. Each lab is designed to simulate real-world network scenarios, focusing on configuration, troubleshooting, and security.

## 📚 Labs Overview

| Lab               | Topic      | Description                          |
| ----------------- | ---------- | ------------------------------------ |
| VLAN Basics       | Switching  | Network segmentation using VLANs     |

1. Download or clone this repository
2. Open any `.pkt` file in Cisco Packet Tracer
3. Review configurations in the `configs/` folder
4. Test connectivity and experiment
